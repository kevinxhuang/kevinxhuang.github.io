ibus是一个很强大的输入，现在主要用于手机和Linux平台上。

下面是ibus日语输入法ibus-anthy的安装方法

``` cpp 
sudo apt-get install ibus-anthy 
```

然后在ibus的首选项里面添加日语->anthy即可
