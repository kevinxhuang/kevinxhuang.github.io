安装

``` bash
sudo apt-get install amarok
```
中文界面
如果安装的Amarok是显示英文菜单，需要安装KDE的中文包才能显示中文菜单。

```
sudo apt-get install language-pack-kde-zh language-pack-kde-zh-base
``` bash
