编辑距离算法 Levenshtein Distance

发表于 2012-01-08 | 分类于 Algorithm

编辑距离，又称Levenshtein距离（也叫做EditDistance），是指两个字串之间，由一个转成另一个所需的最少编辑操作次数，如果它们的距离越大，说明它们越是不同。许可的编辑操作包括将一个字符替换成另一个字符，插入一个字符，删除一个字符。俄罗斯科学家Vladimir Levenshtein在1965年提出这个概念。因此也叫Levenshtein Distance，常用来衡量字符串相似度。
